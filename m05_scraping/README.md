###1.1 Simple web scraper using Python.

`python pl_ceneo_scrap.py -p PRODUCT+NAME` 

Parameters:

`-p` - *product name*

Output:

`PRODUCT+NAME_pl_ceneo_data.csv`

File Header:

`"item_name","item_price","item_category","req_ip","req_country","req_time","req_price_comp"`
